Class Diagram:
- Class: User
    - Attributes: id (int), name (string), email (string)
    - Methods: getUserDetails(), setEmail(email)
    - Relationships: User 1..* Order (A user can have multiple orders)

- Class: Order
    - Attributes: orderId (int), orderDate (date), userId (int)
    - Methods: getOrderDetails(), setOrderDate(orderDate), getUser()
    - Relationships: Order *..1 User (An order belongs to one user)
